# REPeatedOPtogeneticStimulation
Analysis code and processed data for Murano et al bioRxiv 2015 "Repetitive Neuronal Activation Regulates Cellular Maturation State via Nuclear Reprogramming"

## Overview
This repository contains analysis code and processed data for:

Murano et al., bioRxiv (2015)  
"Repetitive Neuronal Activation Regulates Cellular Maturation State via Nuclear Reprogramming"

## Contents
- `scripts/` : Analysis scripts (R / Python)
- `data/` : Processed datasets used for analyses
- `results/` : Generated figures and summary tables

## Data availability
This repository includes processed data sufficient to reproduce the main analyses.
Raw data are not included due to size and availability constraints.

## Reproducibility
Analyses were performed using standard R and Python environments.
Detailed package versions may be added in future updates.
